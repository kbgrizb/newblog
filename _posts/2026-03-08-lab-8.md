I ran into several issues while trying to follow the Razor Pages Web App with ASP.NET Core tutorial.
Most of my issues were pretty minor, though--like I forgot to close the app before trying to run it
again, or I tried building something in the wrong folder. 

It wasn't too difficult to make my own app after completing the tutorial. I decided to do a concert
tracker, since that's something that interests me. It was easier than I thought it would be to add
different fields, and I enjoyed how I could run lines of code in the terminal that would create and fill
out entire files and folders for me. Seeding the data was easy, too. I was able to add a couple examples
that would show up if the table was empty. The Razor framework almost felt easier for me than the
Jekyll framework, but that might be because the tutorial was so in depth. I think I could easily
create another simple Razor app, but I would probably struggle to make it much more advanced.
